# ERA Framework

## 名称
Expectation-Role-Action

## 结构
- **Expectation**: 期望达成的目标
- **Role**: AI 扮演的角色
- **Action**: 需要执行的具体任务

## 适用场景
- 快速提示词构建
- 简单任务指令
- 日常 AI 交互
- 轻量级内容生成
- 初学者入门
- 快速原型测试

## 示例
**输入**: "帮我写代码"

**优化后**:
```
Expectation: 生成一个可运行的 Python 函数，能够计算斐波那契数列
Role: 你是一位经验丰富的 Python 开发者
Action: 编写一个带类型提示和文档字符串的 fibonacci 函数，包含错误处理
```

## 使用提示
- 先明确目标，再定义角色，最后说明任务
- 适合目标明确的简单任务
