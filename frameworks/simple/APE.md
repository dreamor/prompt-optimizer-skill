# APE Framework

## 名称
Action-Purpose-Expectation

## 结构
- **Action**: 具体行动指令
- **Purpose**: 行动目的/原因
- **Expectation**: 期望结果

## 适用场景
- 快速提示词构建
- 简单任务指令
- 日常 AI 交互
- 轻量级内容生成
- 初学者入门
- 快速原型测试

## 示例
**输入**: "写一封邮件"

**优化后**:
```
Action: 写一封正式的商务邮件
Purpose: 向客户说明项目延期的原因并请求理解
Expectation: 邮件应该专业、诚恳，维护客户关系，并获得客户的谅解回复
```

## 使用提示
- 保持每个部分简洁明了
- Action 使用动词开头
- Expectation 应该可衡量或具体
